# Guia: Salvando Alterações no GitHub Desktop

Como você já tem o GitHub Desktop instalado e configurado para este projeto, siga os passos abaixo para salvar as melhorias dos gráficos:

1. **Abra o GitHub Desktop**
   - Certifique-se de que o repositório atual (canto superior esquerdo) é o `dre-marbrasil`.

2. **Verifique as Alterações**
   - Na lista à esquerda, você verá os arquivos que modificamos:
     - `index.html` (Adição dos gráficos e biblioteca de labels)
     - `script.js` (Lógica de cálculo e barras empilhadas)

3. **Realize o Commit (Salvar Localmente)**
   - No canto inferior esquerdo, há um campo para título e descrição.
   - **Título**: `Adiciona gráficos detalhados empilhados no modal`
   - **Descrição** (Opcional): `Implementa gráficos de barras empilhadas com percentuais para detalhamento de custos, despesas e resultado.`
   - Clique no botão azul **Commit to main** (ou master).

4. **Envie para o GitHub (Salvar na Nuvem)**
   - Após o commit, aparecerá um botão ou atalho no topo direito: **Push origin**.
   - Clique nele para subir as alterações para o seu repositório online.

Pronto! Sua versão com os novos gráficos detalhados está salva e segura.
